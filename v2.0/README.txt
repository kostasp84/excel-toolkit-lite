
Excel Toolkit v2.0
==================

English
-------

Excel Toolkit v2.0 is a user-friendly tool for cleaning, merging, grouping, and analyzing Excel and CSV files. It features a modern graphical interface and supports both English and Greek languages.

Features:
- Clean data: Remove duplicates, trim spaces, change case, and target specific columns.
- Merge files: Combine two files based on a common column.
- Group & Aggregate: Group data by a column and apply aggregation functions (sum, mean, count, max, min).
- Statistics: Generate summary statistics for all or selected columns, with optional PDF export.
- Preview Data: View the first 200 rows of your file in a table.
- Multilingual: Switch between English and Greek.

How to Use:
1. Select an action (Clean, Merge, Group, Statistics, Preview Data).
2. Choose File 1 (and File 2 for Merge).
3. Fill in columns and options as needed.
4. Click 'Run' to process and save the result.
5. For help, click the 'Help' button.

Ελληνικά
---------

Το Excel Toolkit v2.0 είναι ένα εύχρηστο εργαλείο για καθαρισμό, συγχώνευση, ομαδοποίηση και ανάλυση αρχείων Excel και CSV. Διαθέτει σύγχρονο γραφικό περιβάλλον και υποστηρίζει αγγλικά και ελληνικά.

Δυνατότητες:
- Καθαρισμός: Αφαίρεση διπλότυπων, κενών, αλλαγή πεζών/κεφαλαίων, στοχευμένες στήλες.
- Συγχώνευση: Συνδυασμός δύο αρχείων με βάση κοινή στήλη.
- Ομαδοποίηση & Άθροιση: Ομαδοποίηση με στήλη και εφαρμογή συναρτήσεων (άθροισμα, μέσος όρος, πλήθος, μέγιστο, ελάχιστο).
- Στατιστικά: Δημιουργία περιγραφικών στατιστικών για όλες ή επιλεγμένες στήλες, με δυνατότητα εξαγωγής σε PDF.
- Προεπισκόπηση: Προβολή των πρώτων 200 γραμμών του αρχείου σε πίνακα.
- Πολυγλωσσικό: Εναλλαγή μεταξύ αγγλικών και ελληνικών.

Οδηγίες Χρήσης:
1. Διάλεξε ενέργεια (Καθαρισμός, Συγχώνευση, Ομαδοποίηση, Στατιστικά, Προεπισκόπηση).
2. Επέλεξε Αρχείο 1 (και Αρχείο 2 για Συγχώνευση).
3. Συμπλήρωσε στήλες και επιλογές όπου χρειάζεται.
4. Πάτησε 'Εκτέλεση' για αποθήκευση αποτελέσματος.
5. Για βοήθεια, πάτησε το κουμπί 'Βοήθεια'.
